A privacy-focused, pure client-side tool to format JSON and convert it to Go Structs.

## Features
- **100% Privacy**: Runs entirely in your browser. No data is sent to any server.
- **Offline Capable**: Works without an internet connection.
- **JSON to Go**: Instantly generate Go struct definitions with json tags from raw JSON.

## How to use
1. Paste JSON.
2. Click "JSON to Go".
3. Copy the struct.
